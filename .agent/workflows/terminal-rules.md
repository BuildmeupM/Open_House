# Terminal Execution Rules (กฎการรัน Terminal)

## กฎสำคัญ: Agent ห้ามรัน Terminal Commands เอง

### สิ่งที่ Agent ทำได้:

1. **เขียนโค้ด** — สร้างและแก้ไขไฟล์ต่างๆ ได้โดยตรง
2. **สรุปคำสั่ง** — บอกผู้ใช้ว่าต้องรันคำสั่งอะไรบ้าง พร้อมอธิบาย

### สิ่งที่ Agent ห้ามทำ:

1. **ห้ามใช้ `run_command`** เพื่อรันคำสั่ง Terminal ใดๆ ทั้งสิ้น
2. **ห้ามติดตั้ง packages** ผ่าน Terminal โดยอัตโนมัติ
3. **ห้ามสั่ง start/stop server** ผ่าน Terminal

### รูปแบบการแจ้งผู้ใช้:

เมื่อ Agent เขียนโค้ดเสร็จ ต้องสรุปคำสั่งที่ผู้ใช้ต้องรันเอง ในรูปแบบนี้:

```
📋 คำสั่งที่ต้องรัน:

1. เปิด Terminal ไปที่โฟลเดอร์ frontend/
   > npm install
   > npm run dev

2. เปิด Terminal อีกตัวไปที่โฟลเดอร์ backend/
   > npm install
   > node server.js
```

### ข้อยกเว้น:

- ไม่มีข้อยกเว้น — ทุกคำสั่ง Terminal ต้องให้ผู้ใช้เป็นคนรันเอง
